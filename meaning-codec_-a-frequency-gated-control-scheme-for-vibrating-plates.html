Meaning Codec A Frequency-Gated Control Scheme for Vibrating Plates

Executive Thesis

A centrally driven square plate with granular tracers implements an ε-gated control surface whose resonant and anti-resonant structure yields lawful drive / break / hold primitives. Eight simple, low-order shape domains (○ → ring → ellipse → arcs → II → hyper-arcs → X → ring-recur) constitute a compact symbol alphabet for a Meaning Codec, exhibiting:

  • Abstention outside eligibility bands ε
  • Sharp causal flips at nearby antiresonances
  • LOW/MDL persistence under coarse-graining and material changes

Core Claims (Falsifiable)

C0 (Abstention law)

Off-band (outside ε) the system does not commit to a stable pattern; abstention rate ≫ 0.5 and time-to-commit diverges with distance from ε.

Red-team: Overdrive amplitude to induce adhesion/friction regime flip. If off-ε spurious "stable" heaps appear persistently, C0 fails → requires safe-amplitude precondition.

C1 (Drive/Break asymmetry)

At the same symbol, a ±1–2 Hz hop from a resonant peak to the nearest antiresonant notch increases time-to-commit and reduces SNR vs a sham hop (Z≥3 vs null).

Red-team: Choose the shallowest possible notch near S₄/S₆. If RES≈ANTI (Δlatency < 1.2×, ΔSNR < 2 dB) reproducibly with good Q, C1 needs a minimum-notch-depth precondition.

C2 (LOW persistence)

Under ×2 temporal downsample or halved drive amplitude, the symbol order (○→…→X→ring-recur) is invariant; higher-index domains thin first.

Red-team: Tweak mounting to make S₆ wider than S₄ persistently (post-fix mounting). If reproducible, C2 needs a boundary-quality precondition.

C3 (Bridge invariance)

Changing plate material/thickness/mass shifts absolute frequencies but preserves (i) symbol order, (ii) ε topology, and (iii) availability/positioning of nearby antiresonant "break" notches (within one ε-band when expressed in units of f/Q).

Red-team: Use P2→P3 with single Q; if |Δδ| > 1.5 systematically, C3 requires piecewise Q(f) correction.

C4 (Controller sufficiency)

A frequency-only controller with RES/ANTI/HOLD semantics achieves deterministic symbol selection with pre-declared accuracy and latency targets; phase and amplitude trims are optional refinements.

C5 (Atypical quarantine)

Narrow, high-brittleness micro-bands (elevated ζ, small Q) either serve as boundary beacons or are excluded without degrading codec completeness.

Symbol Alphabet

Let S = {S₀, …, S₇} with simple, low-order shapes:

k Symbol Human Label Informal Description
0 S₀ ○ filled disk / central mound (lowest-f domain)
1 S₁ ring annulus / nodal center
2 S₂ ellipse elongated ring / symmetry-aligned
3 S₃ arcs two elliptical arcs
4 S₄ II two parallel segments
5 S₅ hyper-arcs two hyperbolic-like arcs
6 S₆ X two diagonals (cross)
7 S₇ ring-recur higher-f ring-like recurrence

Canonical frequency order (increasing f): S₀ → S₁ → S₂ → S₃ → S₄ → S₅ → S₆ → S₇

Note: Not every plate must realize all eight, but the order among realized domains is preserved (LOW).

Core-4

Robust operational subset: {S₁, S₂, S₄, S₆} — ring, ellipse, II (parallel lines), X (diagonals)

Controller Semantics

RES (+)

Park at the resonant peak inside εₖ (or the band center) to commit to Sₖ

ANTI (−)

Hop to the nearest antiresonant notch to erase / break current pattern and reduce commit probability

HOLD (0)

Remain off-ε (abstention) or dwell within ε without switching symbols

These are the only required actions for a working Meaning Codec on the plate; phase and amplitude trims are optional.

Key Metrics & Parameters

Frequency Structures

  • Eligibility band (εₖ = [fₖ⁻, fₖ⁺]): Frequencies where symbol Sₖ is reliably realizable (stability σₖ ≥ 0.8)
  • Interior point (fₖ*): Preferred commit frequency (window center or dominant RES peak)
  • Resonant peak (fₖ⁺): Local maximum of center response; fastest commit
  • Antiresonant notch (fₖ,break): Nearby minimum; best for erase / memory reset

Quality & Damping

Quality factor: Q = fₙ / (fᵣ - fₗ)

Damping ratio: ζ ≈ (fᵣ - fₗ) / (2fₙ) ≈ 1 / (2Q)

Bandwidth: Δf ≈ f / Q

Signals & Metrics

  • Time-to-commit (t_commit): Time from set-frequency to first stable symbol hold (≥ T_hold)
  • Stability (σₖ(f)): Fraction of frames labeled Sₖ during dwell
  • Abstention (a(f)): Fraction of frames labeled "no symbol" (∅)
  • SNR (dB): Pattern-to-background contrast
  • Hysteresis: Up vs down sweep edge difference (Hz)

Core Laws

L1 Abstention

Off-ε → abstain; commits slow/fail outside windows

L2 Drive/Break

RES accelerates commit; ANTI suppresses it (Δlatency, ΔSNR)

L3 LOW persistence

Low-order symbols retain windows and order under coarse-graining; high-order thin first

L4 Bridge invariance

After (f̄, Q) normalization, peaks/notches align across plates (Core-4 accuracy ≥ 90%)

Examples & Walkthroughs

Single-Symbol Commit (clean interior)

Plate: P3 (SUS304, Q≈6.19)

Target: S₆ (X)

ε-window: 172–174 Hz

Procedure:

  1. Erase: set f = 169 Hz (ANTI), dwell T_erase = 0.7 s
  2. Enter: step to f = 174 Hz (center/peak)
  3. Dwell: T_min = 3/Q ≈ 0.485 s
  4. Verify: require σ ≥ 0.8 for T_hold = 0.30 s

Expected: Commit latency median t_commit ≲ 0.58 s; Image shows two diagonals within ±5°; SNR ≥ interior baseline

Monotone 3-Symbol Sequence (LOW-preferred)

Plate: P2 (A5052, Q≈4.24)

Targets: S₁ → S₂ → S₄ (ring → ellipse → II)

Procedure: Sequentially step to each center frequency; dwell 3/Q ≈ 0.71 s; verify each symbol

Expected: All three commits succeed with total wall-time ~2.5–3.0 s

Bridge Transfer (normalized δ) — P2 → P3

Transfer schedules across plates using normalized offset:

δ = (f - f̄ₖ) / (f̄ₖ / Qₖ)

f^(P3) = f̄ + δ · f̄/Q

Expectation: Symbol accuracy ≥ 90% for Core-4 when transferring 10+ such δ-points

  • A1 Vibration: The codec's only control knob is frequency (f=ω/2π); phase is implicit in the plate response.

Core Law: Low-Order Wins

In practice: broader ε, faster commits, and higher stability for lower-index symbols (S₁>…>S₇), and this survives coarse-graining (E4). High-order bands thin first → RG-irrelevant.
